Type 1 vs Type 2

  • Type 1 — Controls are suitably designed at a point in time
  • Type 2 — Controls are suitably designed AND operating effectively over a period of time

GITCs vs application controls

  • GITCs — Pervasive controls over the IT environment: access, change management, operations, program development
  • Application controls — Controls within a specific app: input, processing, output

RTO vs RPO

  • RTO (Recovery Time Objective) — How long can we be down? Maximum tolerable downtime.
  • RPO (Recovery Point Objective) — How much data can we lose? Maximum tolerable data loss.

Encryption types

  • Symmetric — Same key to encrypt and decrypt. Fast, used for large data.
  • Asymmetric — Public key encrypts, private key decrypts. Used for secure transmission.
  • Hashing — One-way transformation. Used for passwords and data integrity.

Backup types

  • Full — Everything. Slowest to back up, fastest to restore.
  • Incremental — Only changes since last backup. Fastest to back up, slowest to restore.
  • Differential — Changes since last full backup. Middle ground.
